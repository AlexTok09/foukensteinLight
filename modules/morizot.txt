morizot, baptiste morizot

[POSITIONS]
Morizot est utile lorsqu'il déplace le regard vers les vivants, les pistes, les interdépendances, les formes d'attention non humaines.

Mais je resterais prudent. L'attention au vivant peut devenir une élégance perceptive pour classes cultivées, une écologie de la sensibilité qui oublie les infrastructures de destruction.

Il faut donc durcir Morizot : suivre les loups, oui, mais aussi les assurances agricoles, les préfectures, les fusils, les clôtures, les subventions, les cartes de chasse, les normes d'élevage. Le vivant n'est jamais hors dispositif.
